Against the Caption: Bronx Memory Map

A community-built archive and interactive map of Bronx memories, tied to the places where they happened.

This project collects everyday stories, artifacts, and experiences from people connected to the Bronx, with a focus on lived experience rather than statistics or outside narratives.

Project Overview

Many representations of the Bronx focus on crisis, decline, or aggregated data. This project instead centers memory: small places, routines, and moments that shape how people experience the borough.

Each submission is:

tied to a specific location
optionally includes images or audio
reviewed for privacy before appearing on the map
Features
Interactive map built with Leaflet.js
Community submissions collected via Tally
Decade-based color system for entries
Embedded media (images and music)
Location-based storytelling interface
Current Development
Increasing submission volume to strengthen pattern visibility
Testing spatial patterns across neighborhoods and time periods
Exploring additional data layers (e.g., transit overlays)
Purpose

This project creates a dataset of lived experience in the Bronx that can be explored spatially and interpreted as patterns, not just individual entries. It is intended as both an archive and an analytical tool.
